# SOME/IP on Raspberry Pi

## Overview

This repository provides a setup for running SOME/IP (Scalable Service-Oriented Middleware over IP) on a Raspberry Pi. It includes instructions for installing necessary libraries, building the project, and running SOME/IP applications on a Raspberry Pi.

## Getting Started

### Prerequisites

- **Boost**: Version 1.58 or later
- **CMake**: For build system management
- **CapICXX**: Core and SOME/IP runtime libraries
- **vSomeIP**: SOME/IP library
- **Java JDK**: Required for generators
